# 📚 Bootcamp DevJr — Módulo 7: Integração Backend + Frontend

## Sobre o Projeto

Aplicação fullstack para gerenciamento de alunos e cursos, desenvolvida como desafio do **Bootcamp DevJr — Módulo 7**.

---

## ⚙️ Backend 

API com os seguintes endpoints disponíveis:

| Método | Rota | Descrição |
|---|---|---|
| `GET` | `/students` | Lista todos os alunos |
| `GET` | `/students/{id}` | Retorna um aluno pelo ID |
| `GET` | `/courses` | Lista todos os cursos |
| `POST` | `/students` | Cadastra um novo aluno |

---

## 🖥️ Frontend

Interface adaptada ao backend com as seguintes funcionalidades:

- Listagem de alunos em tabela
- Cadastro de novo aluno junto ao curso selecionado

---

## 🛠️ Tecnologias

- Java + Spring Boot
- HTML / BootStrap / JavaScript
